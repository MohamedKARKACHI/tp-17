# Comparaison des Formats de Sérialisation

Application Node.js pour comparer les performances de sérialisation entre JSON, XML et Protocol Buffers.

## Fonctionnalités

- Sérialisation et désérialisation de données
- Mesure des temps d'exécution
- Comparaison des tailles de fichiers

## Exécution

```bash
npm install
node index.js
```

## Fichiers générés

- `data.json` - Données au format JSON
- `data.xml` - Données au format XML
- `data.proto` - Données au format Protocol Buffers
